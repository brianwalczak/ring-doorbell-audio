# Ring Doorbell Audio
 
Extracted audio clips from Ring Battery Doorbell firmware (model [5F97F2](https://www.amazon.com/dp/B0BZWRSRWV)), including all setup prompts, alerts, and sound effects.
 
## Contents
 
- Setup flow sounds (wifi connection, pairing, updates, errors)
- Status/notification sounds (recording notice, doorbell chime, booting audio)
- Includes both English and Japanese variants for speech.
